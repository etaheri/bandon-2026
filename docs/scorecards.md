# Bandon Dunes Scorecards — Par & Stroke Index

Research for Bandon Cup '26 scoring engine. Each course must have a complete,
unique stroke-index allocation (values 1..18, no gaps, no duplicates) — this is a
hard requirement for handicap stroke allocation. All five courses below satisfy
that rule.

Stroke index = men's standard handicap allocation per hole.
